# The Rocket Times

A Hacker News frontend made in Astro.

## Build Instructions

1.  ```sh
    git clone https://github.com/noClaps/ZeroLimits.dev
    cd ZeroLimits.dev
    ```

2.  ```sh
    npm install
    npm run dev
    ```

    Then open `localhost:4321` in your browser to see the live preview.

3.  ```sh
    npm run build
    ```

    This will build the site and place the files in `packages/news/dist/`.
