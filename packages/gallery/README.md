# Aperturic Focus

A website to showcase some cool photos I've taken.

## Build instructions

1.  ```sh
    git clone https://github.com/noClaps/ZeroLimits.dev.git
    cd ZeroLimits.dev
    ```

2.  ```sh
    npm install
    npm run dev:gallery
    ```

    Then open `localhost:4321` in your browser to see the live preview of the site.

3.  ```sh
    npm run build:gallery
    ```
    
    This will build the site and place it in `packages/gallery/dist/`.

The images are stored in the [noClaps/assets](https://github.com/noClaps/assets) repository.
